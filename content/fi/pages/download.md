Allekirjoitukset
----------------

Kaikki julkaisut alkaen 2.0.0 versiosta on allekirjoitettu GPG:llä, allekirjoitus
tiedostoilla joihin on linkit ylempänä tällä sivulla. Ne on tehnyt johtava kehittäjä
ja allekirjoitettu hänen yksityksellä avaimellaan jossa on seuraava sormenjälki:

~~~~~~~~~~~~~~~~~~~~~
pub   2048R/BCC8BC4F 2014-08-20 [expires: 2019-08-19]
      Key fingerprint = B1FE 958E D5E8 468E AA20  8F4B F1D8 799F BCC8
      BC4F
uid       [ unknown] Marvin Gülker <m-guelker@quintilianus.eu>
sub   2048R/82C8F74F 2014-08-20 [expires: 2019-08-19]
~~~~~~~~~~~~~~~~~~~~~

Voit löytää julkisen avaimen tämän sivun lopusta (johtuen sen
pituudesta). Eheyden tarkistaminen toimii näin importattuasi
julkisen avaimen:

~~~~~~~~~~~~~~~~~~~~~
% gpg --verify TSC-2.0.0.tar.xz.sig TSC-2.0.0.tar.xz
gpg: Signature made Mon 17 Aug 2015 08:49:18 PM CEST using RSA key ID BCC8BC4F
gpg: Good signature from "Marvin Gülker <m-guelker@quintilianus.eu>" [ultimate]
gpg:                 aka "Marvin Gülker <quintus@quintilianus.eu>" [ultimate]
~~~~~~~~~~~~~~~~~~~~~

Huomioi “good signature” osa joka merkitsee että tiedostoa
ei ole muutettu.

Git pääsy
---------

Projekti [on
GitHubissa](https://github.com/Secretchronicles/TSC) tällä hetkellä. Voit
ladata ja asentaa repon näin:

~~~~~~~~~~~~~~~~~~~
$ git clone git://github.com/Secretchronicles/TSC.git
$ cd TSC
$ git submodule init
$ git submodule update
~~~~~~~~~~~~~~~~~~~

Ole hyvä ja lue ohjeet `INSTALL.md` tiedostosta saadaksesi
lisätietoja.
